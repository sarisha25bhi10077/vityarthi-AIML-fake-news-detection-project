# vityarthi-AIML-fake-news-detection-project
My AIML project is on the topic fake news detection. It is a headline-based fake news detection project which aims to identify misinformation by analyzing only the news title. This approach is highly efficient for real time monitoring on social media platforms .
this model is easier to deploy for the general public and journalists because they require only the text that is visible on the screen.
it uses logistic regression or passive- aggressive classifier to perform best for headlines.
